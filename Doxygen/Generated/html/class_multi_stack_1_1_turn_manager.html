Detailed Description

Handles player turns an spawning of player physics objects objects. Manages and maintains a list of spawned physics objects. Also handles applying certain MultiStack.GameModifier e.g. when MultiStack.ExplosiveShapeModifier is applied, this class will loop through the spawned shape list and convert one shape into an explosive shape.
